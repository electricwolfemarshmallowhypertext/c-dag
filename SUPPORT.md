# Support

## Community/source-available users

Public users can:

- open issues for reproducible defects
- request documentation clarification
- report security concerns through the private channel in `SECURITY.md`

No SLA is provided for public support.

## Commercial licensees

Commercial licensees can receive support under written agreement, including:

- integration guidance
- configuration review
- audit-output review
- pilot governance documentation support

Commercial support does not include legal advice, credit-policy approval, or certified regulatory compliance determinations.
